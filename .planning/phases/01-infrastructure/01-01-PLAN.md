    def _free_path(dst: Path) -> Path:
        """Return dst unchanged if it does not exist; otherwise dst with (2), (3)... appended to stem."""
        if not dst.exists():
            return dst
        stem = dst.stem
        suffix = dst.suffix
        parent = dst.parent
        counter = 2
        while True:
            candidate = parent / f"{stem} ({counter}){suffix}"
            if not candidate.exists():
                return candidate
            counter += 1

<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| user input -> select_drive() | Integer typed at keyboard — could be any string |
| user input -> show_menu() | Menu choice typed at keyboard — could be any string |
| Windows kernel32 API -> Python | ctypes call results depend on OS state (drive letter, permissions) |
| drive filesystem -> Executor.move() | src/dst paths come from caller — must be validated before any mutation |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-01-01 | Tampering | Executor.move() — src/dst paths | mitigate | is_no_touch() and should_skip_path() checked inside move() before any shutil call; call sites cannot bypass |
| T-01-02 | Tampering | _free_path() — destination collision | mitigate | Loop checks dst.exists() before every candidate; never overwrites an existing file |
| T-01-03 | Elevation of Privilege | should_skip_path() — case sensitivity | mitigate | path.lower() applied before substring match; SKIP_PATH_PARTS stored lowercase — case folding prevents bypass via mixed-case paths |
| T-01-04 | Denial of Service | add_file_logging() — PermissionError on read-only media | mitigate | try/except PermissionError wraps mkdir; falls back to console-only logging with a warning; script continues running |
| T-01-05 | Information Disclosure | logging to <drive>\_organizer_logs\ | accept | Log contains only file paths and move results — no credentials, no PII. Drive is user-owned removable media. |
| T-01-06 | Spoofing | select_drive() — user types drive number | mitigate | Input validated: choice.isdigit() AND 1 <= int(choice) <= len(drives); invalid input re-prompts, never indexes out of bounds |
| T-01-07 | Tampering | NO_TOUCH_EXTS bypass via path without extension | mitigate | is_no_touch() returns False for paths without a dot — safe because extension-less files are not ROM/ISO images; the block list is extension-based by design |
| T-01-08 | Denial of Service | Executor.move() — PermissionError / OSError | mitigate | Two-level except catches both; logs error and returns None; calling code receives None and can count/skip; script does not crash |
</threat_model>
